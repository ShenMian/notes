# 绪论

## 定义

**ISO**: 计算机图形学是研究通过计算机将数据转换为图形, 并在专门显示设备上显示的原理/方法和技术的学科.  
**IEEE**: 计算机图形学是利用计算机产生图形图像的艺术或科学.  

## 发展简史

1. 计算机图形学诞生 1950-1960
2. 线框图形学 1960-1970
3. 光栅图形学 1970-1980
4. 真实感图形学 1980-1990

## 渲染管线(图形流水线)

```
应用程序 -> 顶点缓冲区 -> 顶点着色器 -> 光栅化器 -> 片段着色器 -> 帧缓冲区
               CPU -> | <- GPU           |
                                   3D -> | <- 2D
```
