# 游戏环境

## 环境配置

- `nvidia-prime`: 提供 `prime-run` 脚本, 使程序按需调用 NVIDIA 独显.
- `gamemode`: 提供 `gamemoderun` 命令, 优化系统性能.
- `xpadneo-dkms`: Xbox 蓝牙手柄的高级驱动. 解决原生 `xpad` 驱动下可能出现的异常行为 (如游戏退出后手柄持续震动/触发高频异常按键事件等), 并且添加扳机震动支持.

```sh
paru -S nvidia-prime gamemode xpadneo-dkms
```

Steam 游戏启动参数:

```
prime-run gamemoderun %command%
```

## 安装 Proton-GE

```mermaid
graph LR
    A["Wine (社区维护)"]
    A -->|fork| B["Proton (Valve 维护)"]
    B -->|fork| C["Proton-GE (社区维护)"]
```

主要有三种方法:

1. Flatpak (仅适用于 Flatpak 下的 Steam)

    如果你使用的是 Flatpak 版本的 Steam, 可以通过下面命令一步到位:

    ```sh
    flatpak install com.valvesoftware.Steam.CompatibilityTool.Proton-GE
    ```

2. ProtonUp-Qt

    ```sh
    paru -S protonup-qt-bin                       # AUR
    flatpak install flathub net.davidotek.pupgui2 # Flatpak
    ```

    通过 ProtonUp-Qt 为 Steam 安装任意版本的 Proton-GE.

3. 手动安装

    请参考[官方文档](https://github.com/GloriousEggroll/proton-ge-custom#installation).

安装 [Proton-GE] 后即可重启 Steam, 然后在 `Settings | Compatibility | Default compatibility tool` 下拉菜单选择 `GP-Proton` 开头的选项.

[Proton-GE]: https://github.com/GloriousEggroll/proton-ge-custom

## Steam 大屏模式

默认情况下 Steam 的大屏模式 (Big Picture Mode) 存在严重的性能问题, 启用 `Settings | Interface | Enable GPU accelerated rendering in web views (requires restart)` 即可解决.
