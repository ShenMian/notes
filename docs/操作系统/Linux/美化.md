# 美化

## Plymouth

Plymouth 可实现开机动画.

```sh
sudo pacman -S plymouth
```

编辑 `/etc/mkinitcpio.conf` 文件, 在 `HOOKS` 中的 `systemd` 后添加 `plymouth`:

```conf title="/etc/mkinitcpio.conf"
HOOKS=(... systemd plymouth ...)
```

然后重新生成 initramfs:

```sh
sudo mkinitcpio -P
```

详情请参考 [ArchWiki](https://wiki.archlinux.org/title/Plymouth).

## GRUB 主题

[distro-grub-themes](https://github.com/AdisonCavani/distro-grub-themes) 是一个高质量的 GRUB 主题库, 在[此处](https://k1ng.dev/distro-grub-themes/preview)查看效果预览.

### 手动安装步骤

```sh
git clone --depth=1 https://github.com/AdisonCavani/distro-grub-themes.git
cd distro-grub-themes/themes

ls # 查看全部主题

sudo mkdir -p /boot/grub/themes/<THEME>
sudo tar -C /boot/grub/themes/<THEME> -xf <THEME>.tar
```

修改 GRUB 配置文件, 以应用主题:

```conf title="/etc/default/grub"
GRUB_THEME="/boot/grub/themes/<THEME>/theme.txt"
```

重新生成 GRUB 配置:

```sh
sudo grub-mkconfig -o /boot/grub/grub.cfg
```

详情请参考: <https://k1ng.dev/distro-grub-themes/installation#manual-installation>.

!!! warning
    不建议安装 `grub-customizer`, 详情请参考 [Manjaro Linux Forum](https://forum.manjaro.org/t/grub-customizer/172612).
