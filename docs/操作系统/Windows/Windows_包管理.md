# Windows 包管理

常见的 Windows 包管理器有:  

- **[Scoop](Scoop.md)**: 开源 (Unlicense 或 MIT).
- **[WinGet](https://github.com/microsoft/winget-cli)**: 开源 (MIT).
- **[Chocolatey](https://github.com/chocolatey/choco)**: 开源 (Apache-2.0), 商业化 (有付费版本).

## WinGet

```
winget install <PACKAGE>
winget uninstall <PACKAGE>
winget search <PACKAGE>
winget upgrade --all --silent
```
