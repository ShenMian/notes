# Broken Arrow

**中文**: 断箭.  

## 教程

- [For those who aren't military geeks](https://steamcommunity.com/sharedfiles/filedetails/?id=3501205842): 介绍了常见的概念和术语, 可以帮助快速入门.

## 金牌要求

获取金牌的条件需且仅需满足金牌要求, 可以忽略铜牌和银牌要求.
