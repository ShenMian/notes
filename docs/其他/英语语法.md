# 英语语法

## 基本句型

1. 主语 + 不及物**动词**.
2. 主语 + 单及物**动词** + 宾语.
3. 主语 + 双及物**动词** + 间接宾语 + 直接宾语.
4. 主语 + 复杂及物**动词** + 宾语 + 宾语补语.
5. 主语 + 系**动词** + 主语补语 (表语).

其中第 2 种句型也被称为主谓宾, 即主语 + 谓语动词 + 宾语.

## 句子成分

1. **主语 (Subject)**.
2. **谓语动词 (Predicate Verb)**.
3. **宾语 (Object)**.
4. **宾语补语 (Object Complement)**.
5. **主语补语 (表语) (Subject Complement)**.
6. **定语 (Attributive)**: 修饰名词.
7. **状语 (Adverbial)**: 主要修饰谓语动词.
8. **同位语 (Appositive)**.

## 句子类型

- **简单句 (Simple Sentence)**.
- **复合句 (Compound Sentence)**: 简单句的并列. 其中主句 (Main Clause) 包含从句 (Subordinate Clause).
- **复杂句 (Complex Sentence)**: 简单句的嵌套.
