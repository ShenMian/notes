## Edge 插件

!!! info
    部分插件来自 Chrome Web Store, 但可以在 Edge 中正常使用.

!!! warning
    建议在安装只针对部分站点的插件时选定其可访问站点, 以避免对其他站点造成未知影响.

| 名称                      | 描述                                                                           |
| ------------------------- | ------------------------------------------------------------------------------ |
| [PDF Viewer]              | 基于 PDF.js 的 PDF 阅读器.                                                     |
| [Proxy SwitchyOmega 3]    | 手动/自动切换代理.                                                             |
| [Simple Allow Copy]       | 在禁止拷贝内容的网页里拷贝内容. (**开启时可能导致部分网页功能失效**)           |
| [Modern for Hacker News]  | 美化 Hacker News 网站界面.                                                     |
| [Vimium C]                | VIM 快捷键. (**会覆盖部分浏览器快捷键, 进而导致部分页面功能失效**)             |
| [NeatDownloadManager]     | 下载器, 需要结合[本地服务]使用. (仅支持 Windows/macOS, 偶尔无法连接到本地服务) |
| [YouTube™ dual subtitles] | YouTube 双字幕.                                                                |
| [YouTube Adblock]         | 屏蔽 YouTube 视频广告.                                                         |
| [Tampermonkey]            | 插件中的插件, 可以安装更多实用脚本.                                            |

## Tor Browser

请尽量不安装插件, 因为这可能产生更多指纹. 仅建议保留少量安全相关的插件, 比如 NoScript.
