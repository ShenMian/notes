# 列表

Haskell 中的列表是通过**单项链表**实现的.

## 构造运算符

```console
ghci> 1 : 2 : 3 : []
[1,2,3]
```

## 常用函数

```console
ghci> head [1, 2, 3]
1
ghci> tail [1, 2, 3]
[2,3]
ghci> length [1, 2, 3]
3
ghci> init [1, 2, 3]
[1,2]
ghci> null [1, 2, 3]
False
ghci> null []
True
ghci> and [True, False, True]
False
ghci> or [True, False, True]
True
```

## 列表推导(List comprehension)

列表推导类似数学中的集合建构式符号(set-builder notation).

```txt
[ <gen> | <elem> <- <list>, ..., <guard>, ... ]
```

## 习题

### 1

创建函数 `elem`, 如果指定元素在列表中返回 True, 否则返回 False.

```hs
elem :: (Eq a) => a -> [a] -> Bool
```

```hs
elem e [] = False
elem e (x:xs) = (e == x) || (elem e xs)
```

### 2

创建函数 `nub`, 移除指定列表中的全部重复元素.

```hs
nub :: (Eq a) => [a] -> [a]
```

```hs
nub [] = []
nub (x:xs)
  | elem x xs = nub xs
  | otherwise = x : nub xs
```

### 3

创建函数 `isAsc`, 如果指定列表成升序排列返回 True.

```hs
isAsc :: [Int] -> Bool
```

```hs
isAsc [_] = True
isAsc (x:y:xs) = (x <= y) && isAsc (y : xs)
```
