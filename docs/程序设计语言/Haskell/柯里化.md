# 柯里化

**英文**: Currying.

与一般命令式编程语言的函数类型声明不同, Haskell 的函数类型声明并没有特别区分参数与返回值类型.

```hs
min :: Ord a => a -> a -> a
min x y = if x <= y then x else y
```

关键字 `->`[^1] 用于构造函数参数. 该运算符是**右结合**的, 即上面的函数声明可以写成:

```hs
min :: Ord a => a -> (a -> a)
```

所以函数 `min` 的返回类型是一个函数.

```hs
min 1 2
=> (min 1) 2
=> (\y -> if 1 <= y then 1 else y) 2
=> if 1 <= 2 then 1 else 2
=> 1
```

通过命令式语言实现 `min` 函数如下:

```rs
fn min<T: PartialOrd + Copy>(x: T) -> impl Fn(T) -> T {
    move |y| if x < y { x } else { y }
}
```

可以通过 `min(1)(2)` 对其进行调用.

与 C++ 中的 [std::bind](https://en.cppreference.com/w/cpp/utility/functional/bind) 相比, 部分应用(partial application)不能跳过或对参数进行重新排序.

[^1]: <https://wiki.haskell.org/Keywords#-.3E>
