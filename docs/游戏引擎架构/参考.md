# 参考

## 书籍

- [游戏编程模式](https://gameprogrammingpatterns.com/)(Game Programming Patterns): 讲解了一些适用于游戏编程的设计模式.
- [游戏引擎架构](https://www.gameenginebook.com/)(Game Engine Architecture): 讲解了游戏引擎所应该具备的各个模块, 这里参考的是本书的第二版.
- *网络多人游戏架构与编程*: 讲解了网络多人游戏的实现.
