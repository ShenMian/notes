# 绪论

研究对象: 计算.
研究目标: 寻找高效, 低耗的解决方案.

## 计算(Computing)

计算是数据结构与算法研究的对象.  

在这门学科中:

- 计算 = 信息处理
- 计算模型 = 计算机 = 信息处理工具

## 算法(Algorithm)

特定计算模型下, 旨在解决特定问题的指令序列.  

- 输入: 待处理的信息(问题).
- 输出: 经处理的信息(答案).
- 正确性: 的确可以解决指定的问题.
- 确定性: 任意算法都可以描述为一个由基本操作组成的序列.
- 可行性: 每一个基本操作都可以实现, 且在常数时间内完成.
- 有穷性: 对于任何输入, 经有穷次基本操作, 都可以得到输出.
