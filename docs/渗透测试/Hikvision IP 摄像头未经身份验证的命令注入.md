# Hikvision IP 摄像头未经身份验证的命令注入

**英文**: Hikvision IP Camera Unauthenticated Command Injection.  
**CVE 代码**: CVE-2021-36260.

admin:$1$yi$KMvI/d5vTBFIySCw1EjGt0:0:0:root:/:/bin/psh
