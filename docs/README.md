## 构建

### HTML

```sh
pip install mkdocs-material pymdown-extensions mkdocs-git-revision-date-localized-plugin mkdocs-rss-plugin
mkdocs build
```

### PDF

先构建为 HTML 文件, 然后使用浏览器打开指定页面并打印为 PDF 文件.
